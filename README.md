# Laser-Defender
Laser-Defender a game made using Unity2D.

Laser-Defender : Protect mars.
You could Choose any kind of ship you want.
How to Play?
Use arrow keys or wasd keys to move.
Use space bar or left mouse click to shoot at enemies.

Happy playing.
